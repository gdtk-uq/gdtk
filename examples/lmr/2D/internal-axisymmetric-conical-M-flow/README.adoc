= M-flow: Internal, axisymmetric, conical flow
`gdtk/examples/lmr/2D/internal-axisymmetric-conical-m-flow`

Rowan J. Gollan
2025-06-21

This example is of supersonic flow in an M-flow configuration.
M-flow is an internal, axisymmetric, conical flow.
The wall shape supports a conical shock.
This particular case is from the paper by Mölder (1967).
